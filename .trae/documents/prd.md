# 陪诊锦囊 MedPrep — 产品需求文档 v2.0

> 最后更新：2025-06  
> 定位：移动端优先的就诊前—中—后全周期陪诊助手（非诊断工具）

---

## 1. 产品概述

**陪诊锦囊 MedPrep** 帮助患者及家属在就诊前整理症状与问题、解读检查报告、记录诊后医嘱，并通过 PWA 提供用药与复查提醒。

| 维度 | 说明 |
|------|------|
| 目标用户 | 中老年患者及家属 |
| 核心价值 | 就诊信息结构化 + 诊后持续关怀 |
| 数据策略 | 本地优先，用户主动云同步/分享 |
| 合规 | 全功能带免责声明，不提供诊断 |

---

## 2. 功能模块（6 Tab + 账户）

| Tab | 功能 | 数据 |
|-----|------|------|
| 症状时间线 | 中文时间解析 + AI 临床摘要 | localStorage + 云 |
| 问诊清单 | 按科室生成 8–12 条问题 | 同上 |
| 报告解读 | 7 类报告模板 + AI 白话 | 同上 |
| 诊后备忘 | 用药/随访/警示结构化 + AI | 同上 |
| **今日用药** | 用药提醒、复查提醒、到点通知 | IndexedDB + PWA |
| 就诊包 | 汇总、分享、打印、复诊对比 | 同上 |

### 2.1 账户与同步

- 匿名快速体验 / 邮箱登录
- Supabase 云端就诊历史
- 本地未同步记录登录后批量同步

### 2.2 分享

- Supabase 短链 + QR（需登录创建）
- LZ-String 长链降级
- 只读 ShareView

### 2.3 诊后提醒（v2 新增）

- 诊后备忘 → 一键「设为用药提醒 / 复查提醒」
- 今日用药：勾选已服、逾期提示
- Web Notifications + Service Worker（打开 App 时补偿检查）
- 「添加到主屏幕」引导（PWA）

## 3. 页面与路由

| 路由 | 页面 |
|------|------|
| `#/app` | 主工作台（6 Tab） |
| `#/login` | 登录/注册/快速开始 |
| `#/history` | 就诊历史（含离线缓存） |
| `#/history/:id` | 历史详情 + 基于此新建 |
| `#/settings` | 账户设置 |
| `#/print` | 打印视图 |
| `#share=…` | 分享只读页 |

---

## 4. 非功能需求

- 移动端：16px 输入、safe-area、底部 Tab
- 性能：PWA 静态资源缓存
- 隐私：提醒数据默认仅存本机
- 无障碍：支持 prefers-reduced-motion

## 6. 免责声明（产品级）

本工具仅帮助整理就诊信息，不构成医疗诊断或治疗建议。用药与复查提醒为辅助功能，请以医生书面医嘱为准。
